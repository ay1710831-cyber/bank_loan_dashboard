📊 Bank Loan Analysis Dashboard
📌 Project Overview

This project presents an interactive Bank Loan Analysis Dashboard built using Power BI. It helps in analyzing loan applications, approval trends, customer profiles, and financial risks to support data-driven decision-making in the banking sector.

🎯 Objectives
Analyze loan application trends
Track approved vs rejected loans
Identify high-risk customers
Monitor repayment and default rates
Understand customer demographics
📁 Dataset Details

The dataset includes key information such as:

Loan ID
Customer Income
Employment Status
Loan Amount
Loan Purpose
Loan Status (Approved/Rejected)
Interest Rate
Credit Score
Repayment Status
📊 Dashboard Features
🔹 Key KPIs
Total Loan Applications
Total Loan Amount
Approval Rate
Default Rate
🔹 Visual Insights
Loan Distribution by Purpose
Approval vs Rejection Analysis
Customer Segmentation (Income, Job)
Credit Score Impact on Loan Approval
Monthly/Yearly Loan Trends
🔹 Interactive Elements
Slicers for filtering (Region, Loan Type, Time)
Drill-down capabilities
Dynamic charts and visuals
🛠️ Tools & Technologies
Power BI – Dashboard creation
Excel / CSV – Data source
DAX – Calculations & KPIs
Data Cleaning & Modeling
📈 Key Insights
Higher income customers have better approval rates
Certain loan purposes show higher default risk
Credit score plays a crucial role in loan approval
Seasonal trends observed in loan applications

📊 Bank Loan Report Dashboard (Power BI)
📌 Overview

The Bank Loan Report Dashboard is a dynamic Power BI project that provides a detailed analysis of loan applications, funding, repayments, and customer profiles. It includes both summary-level insights and detailed transaction-level data, enabling deeper financial analysis.

🎯 Project Objectives
Analyze loan application trends
Track funded vs received amounts
Evaluate customer risk (Good vs Bad loans)
Understand loan distribution by purpose, term, and demographics
Provide both high-level and detailed insights

📊 Key KPIs
Total Loan Applications: 38.58K
Total Funded Amount: 435.8M
Total Amount Received: 473.1M
Average Interest Rate: 12.0%
Average DTI: 13.3%
📈 Dashboard Features
🔹 Summary Page
KPI cards (Applications, Funded Amount, Received Amount)
Monthly loan trends
Regional distribution (Map)
Loan term analysis (36 vs 60 months)
Loan purpose breakdown
Home ownership insights
🔹 Details Page
Transaction-level loan data
Fields included:
Loan ID
Purpose
Home Ownership
Grade & Sub-grade
Issue Date
Funded Amount
Interest Rate
Installment Amount
🔹 Filters (Slicers)
State
Grade
Good vs Bad Loan
🛠️ Tools & Technologies
Power BI – Dashboard & visualization
DAX – Calculated measures & KPIs
Data Modeling – Relationships
Excel/CSV – Data source
